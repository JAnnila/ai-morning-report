# AI-aamuraportti-arkisto

Tämä repository on automaattisten päivittäisten AI-tiedusteluraporttien arkisto,
jotka Claude Code Routine tuottaa kerran vuorokaudessa.

## Säännöt Claudelle (routine-ajot)
- Tallenna jokainen raportti polkuun `reports/YYYY-MM-DD.md` (yksi tiedosto per päivä,
  ajopäivän mukaan).
- Commitoi raportit SUORAAN `main`-haaraan. ÄLÄ avaa pull requestia äläkä luo
  erillistä haaraa.
- Käytä commit-viestiä muodossa `AI-aamuraportti YYYY-MM-DD`.
- ÄLÄ muokkaa tai poista aiempia raporttitiedostoja.
- Jos jossakin raportin osiossa ei ole vuorokauden aikana merkittävää, kirjoita
  kyseiseen osioon "Ei merkittävää."
